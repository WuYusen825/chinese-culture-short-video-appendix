- Core path model for Conversion: R2 = 0.110; predictors Attraction beta = 0.213, p < .001; Resonance beta = 0.157, p = 0.005; Retention beta = 0.123, p = 0.026.
- Total indirect Attraction-to-Conversion effect through Resonance/Retention = 0.040, bootstrap 95% CI [0.010, 0.077].
